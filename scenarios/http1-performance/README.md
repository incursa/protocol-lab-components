# HTTP/1 Performance Smoke Scenario Pack

This package snapshots the already-committed public `http1.core.plaintext`, `http1.core.json`, and `http1-smoke` contracts from `protocol-lab` commit `a4dcd74e5c8907907ccc58808da92d2b920b2fbc`.

It does not include draft contracts, the broader public HTTP/1 suite, comparison profiles, executor code, implementation code, or evidence. `authority-lock.json` records and validates the exact public-source SHA-256 values.
