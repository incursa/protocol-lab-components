# Support

ProtocolLab Components is a public component package repository. Support here
is limited to package manifests, component wrapper behavior, packaging scripts,
toolchain metadata, and repository documentation.

## Public Questions

Use GitHub issues for:

- component package questions
- package manifest or entry manifest problems
- packaging script problems
- toolchain metadata corrections
- documentation corrections
- new component package proposals

Do not use public issues for private runner operations, hosted lab incidents,
private diagnostics, retained artifacts, credentials, or organization-only
deployment details.

## Security Issues

Report suspected vulnerabilities privately through [SECURITY.md](SECURITY.md).

## Conduct Issues

Report conduct concerns through [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).

## Public Contract Support

Public ProtocolLab schemas, scenarios, suites, load profiles, and contract
fixtures are owned by the `protocol-lab` repository. Component packages in this
repository should conform to those contracts rather than redefining them.
