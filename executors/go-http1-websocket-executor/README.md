# Go HTTP/1.1 cleartext WebSocket executor

`org.protocol-lab.components.executor.go-http1-websocket-executor@0.1.0`
is an exact RFC 6455 client and load generator for the five public cleartext
HTTP/1.1 WebSocket smoke identities. It emits handshake, masking, frame,
payload, clean-close, load, validation, identity, and normalized-result proof.

Every other committed WebSocket identity, including TLS, RFC 8441, RFC 9220,
breadth diagnostics, and `websocket.echo`, exits `3` with
`protocol-lab.unsupported.v1`. Unknown identities exit `2`.
