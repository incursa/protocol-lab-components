# TLS Performance Contract Scenario Pack

This package is an exact snapshot of the committed public TLS v2 scenario family, the smoke and bounded-diagnostic suite/profile routing, the `plab-tls13-p256-v1` profile, and `plab-tls-record-coverage-v1` from ProtocolLab commit `8c4bbe8b7ee94b0e53427dd5ac15e7ede7b77574`.

Executor and target support is independently versioned. Inclusion here recognizes every committed TLS identity but does not imply executable support. The package contains no certificate private key, executor source, implementation source, or benchmark evidence.
