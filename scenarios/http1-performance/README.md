# HTTP/1 Performance Scenario Pack

This package snapshots the committed public `http1.core.plaintext`, `http1.core.json`, `http1-performance-smoke`, `http1-comparison-benchmark`, `http1-smoke`, and `http1-comparison` contracts from `protocol-lab` commit `d5b78d7c07ef0e8a600e92887da2aa150ab89a60`.

It does not include draft contracts, broader HTTP/1 suites, executor code, implementation code, or evidence. `authority-lock.json` records and validates the exact public-source SHA-256 values.
