# TLS Performance Contract Scenario Pack

Package version `0.2.2` adds the authority-owned `tls-security-diagnostics` suite for the existing early-data and KeyUpdate scenarios.

This package is an exact snapshot of the committed public TLS v2 scenario family, the smoke and bounded-diagnostic suite/profile routing, the `plab-tls13-p256-v1` profile, and `plab-tls-record-coverage-v1` from ProtocolLab commit `d5b78d7c07ef0e8a600e92887da2aa150ab89a60`.

Executor and target support is independently versioned. Inclusion here recognizes every committed TLS identity but does not imply executable support. The package contains no certificate private key, executor source, implementation source, or benchmark evidence.
