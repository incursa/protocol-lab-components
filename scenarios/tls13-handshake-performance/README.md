# TLS Performance Contract Scenario Pack

This package is an exact snapshot of the committed public TLS v2 scenario family, `tls-performance-smoke`, `tls-smoke`, and the `plab-tls13-p256-v1` profile fixture from ProtocolLab commit `8c4bbe8b7ee94b0e53427dd5ac15e7ede7b77574`.

The first executable vertical supports only `tls.handshake.full`. Every other included scenario is authoritative recognition material and must be returned as `unsupported` by the initial executor and target. The package contains no certificate private key, executor source, implementation source, or benchmark evidence.
