# MASQUE CONNECT-UDP performance scenario pack

Authority-locked package copy of the public `masque.connect-udp-tunnel`
contract, `masque-connect-udp-comparison` load profile, and
`masque-connect-udp-performance-comparison` non-rankable suite at ProtocolLab
commit `d8082f5ae7d872e9e556f9696177079929483c58`.

The observations are real comparison measurements, but the package does not
claim a rankable topology because client, proxy, and UDP-target roles are not
physically separated.
