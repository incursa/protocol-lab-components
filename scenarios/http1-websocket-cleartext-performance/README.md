# HTTP/1.1 cleartext WebSocket performance scenario package

`org.protocol-lab.components.scenario.http1-websocket-cleartext-performance@0.1.0`
authority-locks the five exact RFC 6455 cleartext identities, smoke suite, and
load profile to public ProtocolLab commit
`8c4bbe8b7ee94b0e53427dd5ac15e7ede7b77574`.

The package does not claim TLS, RFC 8441, RFC 9220, WebTransport, breadth
diagnostics, or the generic `websocket.echo` placeholder.
