# HTTP/1 Performance Smoke Scenario Pack

This package snapshots the committed public `http1.core.plaintext`, `http1.core.json`, `http1-performance-smoke`, and `http1-smoke` contracts from `protocol-lab` commit `0dbb719b29202bf2335c48c03b294035e88839b0`.

It does not include draft contracts, broader HTTP/1 suites, comparison profiles, executor code, implementation code, or evidence. `authority-lock.json` records and validates the exact public-source SHA-256 values.
